La carte des acteurs ne s'affiche pas en ouvrant simplement le fichier html. Voici les façons de voir le site complètement:

1. Ouvrir le fichier sur un serveur local :

-Ouvrir l'invite de commandes Linux ou Bash sur Windows
-Se placer dans le dossier Projet grâce à cd.
-Ecrire python -m SimpleHTTPServer 8080
-Aller sur Google Chrome ou Mozilla Firefox
-Ecrire dans la barre de recherche localhost:8080

Cette méthode est nécessaire pour lire certains scripts (notamment ceux de d3)

2. Aller directement sur le site Internet. Seule la police d'écriture sera différente de celle programmée :
https://etic2017.000webhostapp.com/

Pour 1. et 2., il se peut que certains éléments ne s'affichent pas directement la première fois. Il suffit juste d'actualiser la page avec F5.
